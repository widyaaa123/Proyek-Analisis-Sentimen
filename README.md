## Achievements
Proyek ini berhasil mencapai beberapa pencapaian penting:

- **Data Mandiri:** Dataset yang digunakan merupakan hasil **scraping secara mandiri**.
- **Ekstraksi dan Pelabelan Fitur:** Melakukan tahapan **feature extraction** dan **pelabelan data** untuk memastikan kualitas dataset.
- **Pelatihan Machine Learning:** Menggunakan algoritma machine learning untuk membangun model prediksi.
- **Akurasi Tinggi:** Model yang dibangun berhasil mencapai **akurasi minimal 85%** pada testing set.
